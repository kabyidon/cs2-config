# kabyidon's CS2 Config 

## Installation 
Put `cfg` folder in `C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive\game\csgo`.

## Binds
**Gameplay binds**
- `mwheelup/mwheeldown` jump
- `q` quickswitch (knife)
- `4` cycle grenades
- `x` flashbang/decoy
- `c` smokegrenade
- `v` molotov/incgrenade
- `b` hegrenade
- `h` healthshot
- `j` buymenu
- `mouse4` voice record
- `capslock` toggle voice modenable
- `mouse5` player ping

**Say binds**
- `]` radio1
- `u` team chat
- `6,7,8,9` joke messages
- `f1` vote option1
- `f2` vote option2

**Practice binds**
- `f3` execute practice config
- `f4` give nades
- `f9` execute 1v1 config
- `t` catjumppos (mirage)
- `/` rethrow last nade / preview
- `l` toggle host_timescale 10/1

**Demo playback binds**
- `f5` hold down to fast forward 3x
- `f6` hold down to fast forward 10x
- `f7` toggle pause demo
- `f11` toggle HUD on/off
- `d` disconnect

**Workshop map aliases**
- `aim` Aim Botz
- `fast` Fast Warmup
- `spray` Recoil Master
- `reflex` Fast Aim/Reflex
- `stats` CSStats Training Map
- `rush` Aim_Rush
